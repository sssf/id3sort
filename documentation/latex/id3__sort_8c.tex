Sort the file according to its I\-D3 data. 

Read the files I\-D3 data and move it to the following path\-: \char`\"{}target/artist/album/title.\-mp3\char`\"{}. If no I\-D3 data for artist could be found artist will be \char`\"{}unknown\char`\"{} If album is missing, it will be ignored ex\-: \char`\"{}target/artist/title.\-mp3\char`\"{} If title is missing, the name of the file being processed will be used instead. Target directory must exist.
